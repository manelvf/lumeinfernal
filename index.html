"Lume Infernal" ("Devil's Fire" in Galician) is a MS-DOS game I created in 1997
using the old homebrew C-- compiler. Now you can play it in the browser.

Goal of the game is to "clean" the screen while avoiding two badass enemies.
Enemies speed increases with time. The game has a total of 20 levels.

Gameplay keys:

- Enter: Starts the game and switches to a secondary background.
- Esc: let's you pause the game and leave it if you want. 

Use the mouse to move the game's hero. In previous versions, there was the option to use the arrow keys, but I removed it because most players used only the mouse.

Notice that the background images are not the original.
The original ones were not appropiate, so I replaced them for photographies
of my hometown. So, enjoy.  

Uses DOSBOX.

"Lume Infernal" (Devil's Fire) foi un xogo para MS-DOS que creei en 1997
empregando un linguaxe a medias entre C e ensamblador chamada Sphinx C--.
Agora podes xogalo no navegador.

O obxetivo é limpar a pantalla mentres evitar tocar os dous adversarios
que a patrullan. A velocidade dos inimigos aumenta segundo se avanzan as pantallas.
O xogo ten un total de 20 niveis.

Teclas do xogo:

- Enter: Comeza o xogo e permite ver un fondo secundario.
- Escape: Permite pausar e abandonar o xogo. 

O protagonista móvese co rato. En versións previas, as teclas do cursor podíanse empregar tamén para mover a cara do héroe, mais esa opción foi suprimida porque a maior parte dos xogadores empregaban só o rato.

As imaxes do fondo non son as orixinais de 1997. Tiven que
mudalas xa que o contido das orixinais non era apropiado.
As fotos empregadas agora son da comarca do Ortegal.
